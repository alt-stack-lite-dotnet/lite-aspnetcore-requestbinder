# Conventions (repo)

## Folder layout

- `sources/`: production code (runtime + source generator)
- `test/`: tests (not part of the shipped package)
- `benchmark/`: benchmarks (not part of the shipped package)
- `docs/`: contributor docs
- `docs/wiki/`: user-facing docs

## Documentation rules

- All docs are written in **English**.
- Docs are written in `docs/` and `docs/wiki/`, not inside demo programs.
